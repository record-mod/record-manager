# 💊 ReCord Manager

Application to install and manage ReCord on android.

## ❓ About

ReCord Manager is an application that installs and updates [ReCord](https://github.com/record-mod/record-bundle),
a modification for Discord.

## 💪 Features

Some of the features ReCord Manager provides are:

-   📲 **Install**: Download Discord and install ReCord on it
-   🔁 **Update**: Update ReCord to the latest version
-   🔔 **Notifications**: Be notified when a new version of ReCord is available
-   📃 **Changelog**: See the changelog of ReCord

## 🔽 Download

You can download the most recent version of ReCord Manager from
[here](https://github.com/record-mod/record-manager/releases/latest).  
Learn how to use ReCord Manager by following the [documentation](/docs).
